# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-02-04

### Added
- Initial release of Nanba Payment Gateway
- Smart contract for multi-token payments on Base
- Support for ETH, USDC, and ERC20 tokens
- Platform fee mechanism (2.5% default)
- Refund functionality for merchants
- Backend API with Express.js
- Stripe integration for fiat payments
- Frontend React components
- RainbowKit wallet connection
- Comprehensive test suite
- CI/CD pipeline with GitHub Actions
- Full documentation (README, API, Deployment)

### Security
- ReentrancyGuard on all external functions
- Ownable pattern for admin functions
- Rate limiting on API endpoints
- Input validation on all endpoints
